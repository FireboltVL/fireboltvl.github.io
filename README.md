# Firebolt-VL

**Efficient Vision-Language Understanding with Cross-Modality Modulation**

Firebolt-VL is a novel vision-language model designed for fast multimodal reasoning and fine-grained visual grounding. By replacing heavy cross-attention fusion with a lightweight Cross-modal Modulator (CMM) and state-space modeling, it achieves high performance on consumer-grade hardware.
